1 — Introdução ao Treinamento Scrum

Objetivo

Apresentar o propósito do módulo de treinamento e preparar o usuário para aplicar os conceitos do Scrum de forma prática.

O que é este módulo

Este módulo foi desenvolvido para ajudar você a compreender o Scrum por meio de atividades práticas simuladas. Ao longo das próximas páginas, você aprenderá como funciona um projeto ágil desde o planejamento até a revisão, passando por todas as cerimônias e artefatos do Scrum.

O foco é aprender fazendo — cada seção trará um conceito essencial e uma proposta de atividade simples para colocar em prática o que foi aprendido.

Como funciona

O módulo está dividido em etapas que simulam o ciclo de um projeto Scrum:

  1. O Time Scrum e seus papéis
  2. Criação do Product Backlog
  3. Planejamento da Sprint
  4. Organização do Scrum Board
  5. Simulação da Daily Scrum
  6. Review e Retrospective

Cada etapa contém:

  • Uma explicação breve sobre o tema.
  • Uma atividade prática para realizar individualmente ou em grupo.

Antes de começar

Escolha um projeto exemplo que servirá de base para suas atividades. Pode ser algo simples, como:

  • Um site de receitas,
  • Um aplicativo de lista de tarefas,
  • Um jogo simples,
  • Ou até uma ideia de produto físico.

Esse projeto será usado em todas as etapas do treinamento.

Atividade

Atividade 1 – Escolhendo o projeto:
Pense em um projeto que você gostaria de desenvolver usando Scrum.
Escreva em uma frase curta o nome e o objetivo dele.

Exemplo:
“Aplicativo para organizar tarefas diárias e acompanhar o progresso pessoal.”

Resumo

  • O módulo é composto por várias etapas práticas.
  • Você aprenderá cada parte do Scrum aplicando em um projeto exemplo.
  • O foco é entender o processo, não apenas a teoria.
